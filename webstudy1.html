1주차

1주차의 학습 목표는 html에 대한 강의를 모두 듣고, html에 대한 전반적인 내용을 이해 하는 것이였다. 대부분의 내용은 생활 코딩 카페에 올라와 있는 동영상 강의를 통해 학습할 수 있었지만, 과제를 하는 과정에서 추가적으로 알게 된 내용이 있어 이 페이지에 적어보려한다.

웹에 <> 출력하기

웹페이지에 태그의 문법에 해당하는 <>을 그대로 출력하기 위해서는 < 표시는 & + l + t를 이용하여 나타낸다. 또한 >표시는 & + g + t 를 이용하여 나타낸다.

table tag를 이용한 표만들기

표를 만들기 위해서 사용되는 대표적인 태그는 크게 4가지 이다.

  • table tag (<table> .... </table>) : 테이블을 만드는 태그
  • th tag (<th> .... </th>) : 테이블의 첫번째 행 즉, 속성들을 적는 태그
  • tr tag (<tr> .... </tr>) : 테이블의 행을 만드는 태그
  • td tag (<td> .... </td>) : 테이블의 열을 만드는 태그
* 참고 페이지 : 코딩 팩토리
